Atlas Command Line Interface (CLI)

Atlas CLI allows you to build Atlas projects from your command line. Someday we'll make a real installer, but for now:

  1. Download the binary for your OS (x86 architectures):
    • OS/X
    • Linux
    • Windows (buggy)
  2. Put the file somewhere on your main path
  3. For Mac or Linux, do "chmod +x"
  4. Report questions or issues
